# Unidad 4 — Modelo de Programación Lógica

**TecNM Tijuana — Ingeniería en Sistemas Computacionales**
**Programación Lógica y Funcional**

---

## Objetivo

Desarrollar programas lógicos de complejidad media-alta usando metodología formal,
estrategias de búsqueda avanzadas, programación por restricciones (CLP/FD) y
proyectos multi-paradigma que integran todo lo aprendido en el semestre.

---

## Temas

| Tema | Contenido | Herramienta principal |
|------|-----------|----------------------|
| 4.1 | Desarrollo de programas lógicos con módulos | SWI-Prolog modules |
| 4.2 | Estrategias de búsqueda: DFS, BFS, cut | SWI-Prolog + trace |
| 4.3 | Optimización: CLP(FD), restricciones, Sudoku | library(clpfd) |
| 4.4 | Proyecto práctico multi-paradigma | Prolog + Erlang + Haskell |

---

## Requisitos previos

- SWI-Prolog 9.x instalado: `swipl --version`
- Haber completado Unidad 3 (predicados, backtracking, listas)
- Erlang/OTP 26+ para el proyecto final

| Rubro | Peso |
